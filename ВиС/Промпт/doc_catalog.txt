# Перечень документов

0. Выписка по счёту карты (Дебетовая карта)
Примечание: Может включать детализацию по категориям расходов. Подходит для визы. Доступна даже для закрытой/заблокированной карты.
Примеры запросов: "выписка с дебетовой карты", "движение средств по дебетовой карте", "зарплатные поступления на дебетовую карту"

1. Выписка по счёту карты (Кредитная карта)
Примечание: Выдается только по кредитной карте. Может включать детализацию по категориям расходов. Подходит для визы. Доступна даже для закрытой/заблокированной карты.
Примеры запросов: "выписка по кредитке", "выписка по счету кредитной карты", "движение средств по кредитной карте"

2. Выписка по вкладу или счёту
3. Справка о доступном остатке
Примечание: Подходит для визы, выдается на русском и английском языке. Выдает информацию по остаткам сразу на всех счетах и картах.
Примеры запросов: "справка об остатке", "справка для визы", "справка об остатке на английском языке"
